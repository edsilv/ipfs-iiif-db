# ipfs-iiif-annotations

> IIIF annotations JS client over IPFS

# Instructions

## Install

## Include

In your HTML:

```html
<script src="..."></sript>
```

## Use

### start


```js
iia.start([callback])
```

### change a value

```js
iiia.put(id, value, callback)
```

The id needs to be a string, but the value can be any JS object that can be represented in JSON.


### stop


```js
iia.stop([callback])
```
